    • Von Neumann structure

      Also called a Princeton architecture, is a memory structure that combines program instruction memory and data memory. The program instruction memory address and the data memory address point to different physical positions in the same memory.

    • Harvard Structure

      The Harvard architecture is a memory structure that separates program instruction storage from data storage. Its main feature is to store programs and data in separate storage spaces for independent addressing.

      browse Share

    • Improved Harvard Structure

      The improved Harvard architecture has an independent address bus and an independent data bus. The two buses are time-multiplexed by program memory and data memory to make the structure more compact.
